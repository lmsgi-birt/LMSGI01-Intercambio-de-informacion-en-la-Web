2.3.6.1.- Ejemplo.

Un ejemplo de un documento HTML que muestra el modo de utilizar algunos de estos elementos es: 

<!DOCTYPE html>
<html>
<head>
   <title>Tablas</title>
</head>
<body>
   <h3>Ejemplo de tablas en HTML </h3>
   <table>
     <tr>
       <th>Nombre</th>
       <th>Apellido</th> 
       <th>Edad</th>
     </tr>
     <tr>
       <td>Maria</td>
       <td>Agirre</td> 
       <td>50</td>
     </tr>
     <tr>
       <td>Martin</td>
       <td>Aranburu</td> 
       <td>94</td>
     </tr>
   </table>
</body>
</html>
